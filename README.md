# Train Route Finder Coursework
This is my coursework for the Object Oriented Programming module.

## Launching the program

1. Unzip the provided folder to a location of your choice.
2. Make sure that you have the latest version of Java installed. It can be downloaded from here: https://java.com/en/download/

To run via the file explorer program:

3. Double click on the trf.jar file. The program should open up.

To run via the command line:

3. Open up the command line of your choice.
4. Navigate to the Train Route Finder folder.

To run the .jar file:

5. Use the command "trf.jar".

To run the class files:

5. Use the command "java -classpath bin train.Main".

This last step assumes that you have Java added to your PATH.
